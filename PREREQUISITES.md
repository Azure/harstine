# Pre-requisites for enabling Project Harstine
To set up the private preview for Project Harstine. You will need to do the following: 
1.	Access to the Private Preview documentation (wahoo! You have already completed this step). 
2.	Contact the Dev Box engineering team to enable the Private Preview for you (wahoo! This step is already complete if you have access to this documentation). 
3.	Windows 10/11 Enterprise license, Microsoft Intune License, and a Microsoft Entra ID P1 License. Configure Microsoft Intune and Entra. 
4.	At least 1 W365 Frontline license.
a.	If you would like to test out the preview with higher SKUs/ more licenses than the trial license, please contact support. 
5.	A tenant where you are a tenant-level admin. 
a.	If you are using an older tenant (vs a new tenant), ensure that you increase your Entra device limits to create Dev Boxes. 
6.	A new Azure subscription with Owner and Contributor access. 

Once you have copmpleted the pre-requisites, you can now move onto Dev Bod Set Up. Please review the instructions in setup.md.
